# 🛡️ Web Vulnerability Cheatsheets

A structured and practical collection of web security vulnerability cheatsheets designed for **penetration testers**, **bug bounty hunters**, and **security researchers**.

---

## 📌 About

This repository contains well-organised notes and practical references covering the most common and impactful web vulnerabilities.

Each cheatsheet includes:

* 📖 Clear explanation
* ⚡ Real-world attack techniques
* 🧪 Example payloads
* 🔍 Detection methods
* 🔐 Prevention tips

## 📚 Covered Vulnerabilities

### 🔓 Broken Access Control

* IDOR (Insecure Direct Object Reference)
* Privilege Escalation
* Access Control Issues

### 🔑 Authentication Vulnerabilities

* Reset Password Issues
* 2FA Bypass
* Broken Session Management
* Rate Limiting Issues
* JWT Attacks

### 💉 Injection Attacks

* SQL Injection
* NoSQL Injection
* Command Injection
* SSTI (Server-Side Template Injection)
* XPath Injection
* LDAP Injection
* CRLF Injection

### 🌐 Client-Side Vulnerabilities

* Cross-Site Scripting (XSS)
* CSP Bypass
* CSRF (Cross-Site Request Forgery)
* CORS Misconfiguration
* DOM-Based Attacks
